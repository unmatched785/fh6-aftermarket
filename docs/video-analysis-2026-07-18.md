## 상태 머신에 반영할 결론

지도 필터를 닫은 직후 바로 `X`를 보내면 안 됩니다.

```text
필터 닫기
→ 반투명 아이콘 검출
→ Esc로 지도 닫기
→ 지도 닫힘 검출
→ Enter로 지도 다시 열기
→ 선명한 아이콘 검출
→ 아이콘 중심으로 마우스 이동
→ Enter
→ 정보 카드 검출
→ X
→ 빠른 이동 확인창 검출
→ Enter
→ 도착 화면 검출
```

아이콘은 필터 종료 직후에는 반투명 상태입니다. `Esc`로 지도를 닫고 `Enter`로 다시 연 뒤
선명한 아이콘을 확인해야 마우스 이동을 시작합니다. 이후 아이콘에서 `Enter`를 눌러 정보
카드가 보이는 상태를 실제 선택 완료 상태로 사용합니다. 카드가 열리지 않으면 마우스 위치가
어긋난 것으로 판단하고 안전 정지해야 합니다.
