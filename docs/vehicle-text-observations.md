# 차량명 화면 관찰

2026-07-18에 제공된 여섯 장의 실제 등장 화면을 기준으로 작성했습니다. 원본 이미지는
다양한 해상도와 잘린 비율을 포함하고 개인정보도 표시하므로 Git에 넣지 않습니다.

## 화면에서 확인한 축약 방식

- `Aventador '12`
- `Lambo Sesto`
- `Diablo GTR`
- `F8 Tributo '19`

차량명은 KOR 화면에서도 라틴 문자로 표시되므로 1차 OCR 언어는 영어만 사용합니다.
정식명뿐 아니라 축약명을 모두 목표 사전에 넣습니다.

## 비목표 텍스트 회귀 사례

같은 화면에 `Urus '19`, `Ferrari F12tdf` 같은 비목표 차량도 함께 표시될 수 있습니다.
이 텍스트는 목표로 판정되면 안 됩니다.

## 안전 판정

- 목표 별칭이 정확히 또는 높은 유사도로 잡힘: 즉시 정지
- 텍스트가 읽혔지만 목표 없음: 아직 재시작 허용 안 함
- 보이는 판매 배너 수와 OCR로 읽은 차량명 수가 일치함: 이후 `CLEAR` 후보
- 일부 배너가 가려짐, OCR 누락, 화면 비율 불명: `UNCERTAIN`으로 정지

전체 차량 목록이 없어도 목표 탐지는 가능하지만, 자동 재시작을 허용하려면 화면에 보이는
모든 판매 배너가 판독됐다는 별도 완전성 검사가 필요합니다.
