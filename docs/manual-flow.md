# 검증된 수동 흐름

기준 해상도는 1920×1080입니다. 최초 순서는 사용자가 2026-07-18에 직접 확인했고,
KOR(시스템 언어)→English US 흐름은 2026-07-19에 QHD 실화면에서 키마다 다시
확인했습니다.

## GUI 실전 사이클 시작점

GUI의 첫 시작점은 언어와 관계없이 **오픈월드 정차 상태**입니다. Start menu나 지도 안에서
F1을 누르지 않습니다. 현재 실전 1회 빌드는 정확한 FH6 전경과 16:9 화면을 확인한 뒤
`M`으로 월드 맵을 바로 엽니다. Evolving World 지구본으로 빠른 이동한 뒤 차량을 움직이지
않고, 다시 `M`으로 지도를 열어 Aftermarket Cars 필터와 최대 확대를 진행합니다.

## 한국어에서 English US로 변경

```text
Esc
→ Down ×1
→ Right ×1
→ Enter ×1
→ Up ×2
→ Enter ×1
→ Down ×5
→ Enter ×1 (English US 선택)
→ Down ×1
→ Enter ×1
→ 클라이언트 재시작
```

2026-07-19 현재 이 PC의 시작 언어 항목은 `시스템 언어`이며 Windows 설정에 따라
게임 UI가 한국어로 표시됩니다. `시스템 언어`에서 `English US`까지는 `Down ×5`입니다.
여기서 `Enter`로 언어를 선택하면 재시작 확인창이 열리고 기본 포커스는 `아니요`입니다.
그 뒤의 `Down ×1`은 `예`로 이동하는 별개의 확인 동작입니다. 마지막 `Enter` 뒤 기존
Forza 창이 종료되고 새 창으로 재생성됐으며, 새 시작 화면의 `Enter Start Game`과
`Accessibility/Settings` 문구로 English US 적용을 확인했습니다.

## English US에서 한국어로 변경

```text
Esc
→ Down ×1
→ Right ×1
→ Enter ×1
→ Up ×2
→ Enter ×1
→ Up ×4
→ Enter ×1 (한국어 선택)
→ Down ×1 (Yes 선택)
→ Enter ×1 (재시작 확정)
→ 클라이언트 재시작
```

`Down ×1`의 System Language는 Windows 표시 언어에 따라 결과가 달라질 수 있으므로 사용하지
않습니다. 제공된 언어 목록에서 English US에서 명시적 한국어까지 `Up ×4`임을 기준으로
이동합니다. 언어 선택 뒤 확인창은 `No`가 기본이므로 `Down ×1 → Enter ×1`로 확정합니다. 현재
설정의 `automationReady`는 `false`입니다.

## 재시작 후 필터링된 지도까지

```text
Enter ×1
→ 로딩
→ Enter ×1
→ 로딩
→ Esc ×1
→ 로딩
→ Esc ×1
→ Enter ×1
→ PageDown ×1
→ Enter ×1
→ Enter ×2
→ Down ×4
→ Enter ×1
→ Esc ×1 (필터 닫기)
→ 반투명 애프터마켓 아이콘 확인
→ Esc ×1 (지도 닫기)
→ Enter ×1 (지도 다시 열기)
→ 선명한 애프터마켓 아이콘 확인
```

첫 번째 `Esc` 뒤에는 지도 필터의 `Evolving World`만 선택되고 애프터마켓 아이콘이
반투명하게 표시됩니다. 여기서 `Esc`로 지도를 완전히 닫은 다음 `Enter`로 다시 열어야
아이콘이 선명하게 표시됩니다.

## 애프터마켓 장소 열기

```text
지구본 아이콘 형태 감지 (색상 무관)
→ 아이콘 중심으로 마우스 이동
→ X ×1
→ 빠른 이동 확인창 확인
→ Enter ×1
→ 도착 화면 확인
```

마우스 위치는 고정 픽셀로 저장하지 않습니다. 지도 재진입 뒤 선명한 아이콘을 감지한 후 그 중심점을
사용하며, 아이콘이 하나만 확실하게 검출되지 않으면 입력하지 않고 정지합니다. 아이콘에
커서를 놓은 뒤 `X`와 확인 `Enter`로 빠른 이동합니다.

## 전체 지도에서 판매 차량 3대 확인

```text
애프터마켓 도착 확인
→ 차량 이동 없이 M ×1 (전체 지도)
→ PageDown ×1 (지도 필터)
→ All 행 포커스 확인
→ Enter ×2 (전체 필터 해제)
→ Down ×26 (실화면 기준 Aftermarket Cars)
→ Aftermarket Cars 문구 확인
→ Enter ×1 (활성화)
→ 체크 상태 확인
→ Esc ×1 (필터 닫기)
→ 빠른 이동으로 대리점과 차량 군집이 플레이어 화살표에 겹친 상태인지 재확인
→ Up 길게 누르기 (빠른 이동 때 남은 고정 커서 위치 중심으로 최대 확대)
→ 초록색 판매 차량 아이콘 정확히 3개 확인
→ 화살표를 피해 군집의 1시 방향부터 세 아이콘을 하나씩 호버해 차량명 카드 OCR
→ FH6 Meta 공식 차량 ID로 정규화
→ 여섯 한정 차량 ID와 대조
```

애프터마켓에서 멀리 떨어져 있으면 판매 차량 아이콘 자체가 지도에 나타나지 않으므로,
대리점으로 빠른 이동한 상태가 필수입니다. 축소 상태에서는 초록색 차량 아이콘이 보이지
않으며, 위쪽 방향키를 유지하면 마우스 커서 위치를 중심으로 빠르게 최대 확대됩니다.
정확한 유지 시간을 측정하기 전까지 이 흐름의 `automationReady`는 `false`입니다.
아이콘이 정확히 3개가 아니거나 카드 하나라도 읽히지 않으면 결과를 확정하지 않고 멈춥니다.

세 카드 중 목표 차량이 확인되면 경고음을 세 번 재생하고 추가 입력 없이 정지합니다. 서로
다른 비목표 공식 차량 ID 3개가 확인되면 지도를 `Esc`로 닫고 사용자 제공 English US→한국어 키 순서를
실행해 마지막 `Enter`로 클라이언트 재시작을 시작합니다. 새 클라이언트에는 아직 추가 입력을
보내지 않습니다.
