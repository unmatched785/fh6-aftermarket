# 검증된 수동 흐름

기준 해상도는 1920×1080이며, 아래 순서는 사용자가 2026-07-18에 직접 확인한
수동 입력을 그대로 기록한 것입니다. 아직 자동 입력의 성공을 의미하지는 않습니다.

## 한국어에서 English US로 변경

```text
Esc
→ Down ×1
→ Right ×1
→ Enter ×1
→ Up ×2
→ Enter ×1
→ Down ×6
→ Down ×1
→ Enter ×1
→ 클라이언트 재시작
```

언어 목록 화면에서는 한국어가 목록 상단에 있고, English US가 일곱 번째 표시 위치에
있음을 확인했습니다. 마지막 `Down ×1`은 언어 목록 이동과 별개의 재시작 확인 동작으로
취급합니다.

## English US에서 한국어로 변경

```text
Esc
→ Down ×1
→ Right ×1
→ Enter ×1
→ Up ×2
→ Enter ×1
→ Up ×4
→ Down ×1
→ Enter ×1
→ 클라이언트 재시작
```

`Up ×4 → Down ×1`의 의미는 실제 ENG 화면 캡처로 한 번 더 확인하기 전까지
세부 메뉴 의미를 추정하지 않고, 검증된 입력 순서 그대로 보존합니다.

## 재시작 후 필터링된 지도까지

```text
Enter ×1
→ 로딩
→ Enter ×1
→ 로딩
→ Esc ×1
→ 로딩
→ Esc ×1
→ Enter ×1
→ PageDown ×1
→ Enter ×1
→ Enter ×2
→ Down ×4
→ Enter ×1
→ Esc ×1 (필터 닫기)
→ 반투명 애프터마켓 아이콘 확인
→ Esc ×1 (지도 닫기)
→ Enter ×1 (지도 다시 열기)
→ 선명한 애프터마켓 아이콘 확인
```

첫 번째 `Esc` 뒤에는 지도 필터의 `Evolving World`만 선택되고 애프터마켓 아이콘이
반투명하게 표시됩니다. 여기서 `Esc`로 지도를 완전히 닫은 다음 `Enter`로 다시 열어야
아이콘이 선명하게 표시됩니다.

## 애프터마켓 장소 열기

```text
선명한 보라색 지구본 아이콘 감지
→ 아이콘 중심으로 마우스 이동
→ Enter ×1
→ 애프터마켓 정보 카드 확인
→ X ×1
→ 빠른 이동 확인창 확인
→ Enter ×1
→ 도착 화면 확인
```

마우스 위치는 고정 픽셀로 저장하지 않습니다. 지도 재진입 뒤 선명한 아이콘을 감지한 후 그 중심점을
사용하며, 아이콘이 하나만 확실하게 검출되지 않으면 입력하지 않고 정지합니다. 아이콘에
커서를 놓은 뒤 `Enter`를 눌러 정보 카드가 열린 것을 확인해야만 `X` 빠른 이동을 허용합니다.
